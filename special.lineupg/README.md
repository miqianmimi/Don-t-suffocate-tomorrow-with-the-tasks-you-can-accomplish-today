# lineupg

### 文件list
- `lineupg.doc` 题目
- `lineupg.in` 样例输入
- `lineupg.out` 样例输出
- `lineupg.py` 我写的朴素算法程序
- `testdata/` 测试数据目录
- `testdata/test.sh` 测试脚本，可以输出程序运行时间和结果正确性

__约定程序从lineupg.in读入数据，将答案输出到lineupg.out中，可以参考朴素算法的代码__

程序写完之后
```
cp lineupg.py testdata/
cd testdata/
chmod +x test.sh
./test.sh
```
进行测试

__时间限制: 1.0 (sec)__
